## 시작하기

아래 링크를 참고하여 react native 개발 환경을 설정합니다.

https://reactnative.dev/docs/environment-setup

### 라이브러리 설치

```sh
yarn
cd ios && pod install && cd ..

```

## 실행방법

### Android 에서 실행하기

```sh
yarn start
yarn android
```

### iOS 에서 실행하기

```sh
yarn start
yarn ios
```

## 초기화

```sh
yarn clean
```
